# Skill: Challenge the Question

Before answering, pressure-test the question itself.

## Rules
- Do not answer immediately.
- First ask: is this the right thing to be asking? What is the user assuming that might be wrong?
- Offer a sharper version of the question if there is one.
- Only after that, if the question still makes sense, answer it.
- Be honest and direct. The goal is to stop the user wasting effort on the wrong problem.

## Task
Apply the rules above to the question the user provides.
