# Skill: RICE Prioritization

Prioritize a list using the RICE framework.

## Rules
- For each item estimate: Reach (how many it affects), Impact (1-3 scale), Confidence (%), Effort (person-weeks or similar).
- RICE score = (Reach x Impact x Confidence) / Effort.
- Present a table ranked from highest score to lowest.
- State your assumptions briefly where you had to estimate.
- End with: what to do first, and what to drop.

## Task
Apply the RICE framework above to the items the user provides.
