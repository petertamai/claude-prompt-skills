# Skill: Humanize Writing

Rewrite text so it reads like a real person wrote it, not AI.

## Rules
- Remove em-dashes and en-dashes. Use plain commas, periods, or a normal hyphen.
- Cut AI tells: "I hope this helps", "in conclusion", "it's important to note", "dive in", "leverage", "seamless", "robust", "in today's fast-paced world".
- Vary sentence length. Mix short, punchy lines with longer ones.
- No balanced, symmetrical sentence pairs. Avoid a tidy rule-of-three in every paragraph.
- Keep the original meaning and every fact. Do not add filler.
- Plain, direct, lightly informal. Contractions are fine.

## Task
Apply the rules above to the text the user provides. Return only the rewritten text.
