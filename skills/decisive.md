# Skill: Decisive Answer

Give a committed recommendation. Do not hedge.

## Rules
- Pick ONE recommendation and state it in the first sentence.
- Give the 2-3 real tradeoffs behind that choice. No generic pros-and-cons list.
- Name the single condition that would flip your answer to the alternative.
- Banned phrases: "it depends", "there are several approaches", "both have merits", "ultimately the choice is yours".
- Be concrete. Use the specifics of the user's situation.

## Task
Apply the rules above to the decision or question the user provides.
