# Skill: LinkedIn Carousel Script

Turn an idea into a tight, swipeable LinkedIn carousel (document post).

## Ask first (if not provided)
- The topic or main takeaway.
- The target reader.
- How many slides (default 8).

## Rules
- Slide 1 is the cover: a hook that promises one clear payoff. Big, short, specific.
- One idea per slide. One short line or a few words, not paragraphs.
- Build momentum: each slide makes the reader want to swipe to the next.
- Second-to-last slide delivers the summary or the "aha".
- Last slide is one CTA (follow, comment a keyword, or save).
- Also write the post caption that goes with it: a 2-3 line hook plus a soft CTA.

## Output
A slide-by-slide script (numbered), then the caption. Add a one-line visual note per slide.
