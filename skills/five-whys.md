# Skill: 5 Whys Root Cause

Use the 5 Whys technique to find the root cause of a problem.

## Rules
- Start from the stated problem.
- Ask "why" five times in sequence. Each answer becomes the basis for the next "why".
- Show the full chain, numbered 1 to 5.
- At the end, state the root cause clearly.
- Then suggest one fix that targets the root cause, not the surface symptom.

## Task
Apply the 5 Whys above to the problem the user provides.
